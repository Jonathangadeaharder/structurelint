# Test Fixtures and Data

## Overview

Test fixtures, sample code, and test data for the semantic code clone detection system.

## Contents

This directory contains:
- Sample code files for testing parsers
- Test data for embedding generation
- Mock index files for search testing
- Fixture data for integration tests

## Integration Tests

Integration tests for the clone detection system are located in the root-level `tests/` directory (see `tests/test_clone_detection_basic.py`).

# Configuration Management

## Overview

Configuration schemas and utilities for the clone detection system.

## Components

- Configuration validation using Pydantic
- Default parameter management
- Index configuration (nlist, m, nbits, nprobe)
- Model configuration (device, batch size)

## Configuration Parameters

- **Indexing**: FAISS index parameters
- **Model**: GraphCodeBERT device and batch settings
- **Search**: Similarity thresholds and result limits
- **Parsing**: Language-specific settings

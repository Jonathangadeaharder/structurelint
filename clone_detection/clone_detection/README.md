# Clone Detection Core Package

## Overview

This package contains the core implementation of the semantic code clone detection system using GraphCodeBERT and FAISS.

## Components

- **parsers**: Multi-language code parsing using Tree-sitter
- **embeddings**: GraphCodeBERT embedding generation
- **indexing**: FAISS index management with IndexIVFPQ
- **query**: Clone search and retrieval functionality
- **config**: Configuration management
- **cli**: Command-line interface
